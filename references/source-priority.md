# Source Priority

Use the highest-authority sources available and downgrade carefully.

## Preferred Source Order

1. official annual reports, interim reports, quarterly reports, 20-F/10-K/10-Q equivalents
2. regulator-hosted filings and exchange announcements
3. company earnings presentations, earnings call transcripts, investor-day decks
4. official company operating data releases
5. rating reports, regulator statistics, and audited subsidiary filings
6. broker research or reputable financial media, only as secondary support

## Rules

- Build the numerical base from official filings whenever possible.
- Use secondary sources only to accelerate discovery or to locate the official filing.
- If a source is unofficial but appears to reproduce official numbers, still try to trace it back to the filing.
- When the latest filing is missing, say that clearly instead of silently substituting another source.

## Direct Quote Policy

- Keep direct quotes short.
- Prefer paraphrase plus a link to the filing.
- For controversial claims, quote only the minimum language needed and attribute it precisely.

## Attribution Pattern

Use this style in analysis:

- `Fact:` sourced from the filing or official release.
- `Inference:` your interpretation based on one or more facts.
- `Gap:` data not available or not yet verified.

## When Sources Conflict

- prefer audited over unaudited
- prefer final annual report over preliminary earnings release
- prefer consolidated over parent-only, unless parent-only is the relevant legal entity
- prefer the latest restated numbers over earlier original numbers

When a company restates prior periods, use the restated numbers and note that comparability changed.
